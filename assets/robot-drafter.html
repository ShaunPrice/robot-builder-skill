Robot file (URDF)

This is a URDF — the standard file that describes a robot's shape and joints. Save it into a ROS 2 workspace and the same file drives the 3D simulators (Gazebo, RViz) and the Nav2 navigation stack, so your design carries straight into simulation. It's an approximate starting point — adjust the weights for your real parts.
